## 사용된 기술

## 기능

__익명 채팅과 익명 화상 통화가 가능합니다.__

__익명 채팅은 각 방마다 최대 10명만 입장 가능합니다.__
__화상 채팅의 경우 네트워크의 종류에 따라 연결 불가능한 네트워크가 존재합니다.__
(유동IP 이슈)
